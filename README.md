# Pannash

A minimalist anti-CRM sales tool for individual salespeople.

## Core Vision

- Connect email (Gmail/Outlook) + phone (Twilio)
- Upload CSV of leads
- AI generates ONE strong initial outreach template
- Bulk sends with human-like pacing (30-300s delays, business hours only)
- Smart follow-ups triggered by real engagement
- AI detects hot leads → human takes over
- Ultra-minimal: automation that disappears when it matters

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Database**: Supabase/PostgreSQL
- **Deployment**: Vercel
- **Payments**: Stripe
- **AI**: OpenAI/Claude
- **Email**: Resend/SendGrid
- **SMS**: Twilio

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
